How to for frappe V5:

1. Click on icon of Jasper Erpnext Report then in Jasper Reports an select New.
2. Give it a name; for instance: "Empty Data Source Report Local". The name must be unique.
3. Select in <<Choose Origin>> -> LocalServer.
4. Select in <<Report for>> -> General.
5. Uncheck the <<Use For Custom Fields>> checkbox.
6. Seve it.
7. Click in Upload button.
8. Import EmptyDataSourceReport.jrxml.
9. Save again.
10. Reload/Refresh.
11. Go to desk.
12. Click in the icon of jasper in the upper-right corner.
13. Select the report you just created clicking on pdf icon for instance to make a pdf version.
14. That's all.
